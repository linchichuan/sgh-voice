1. はじめに

本プライバシーポリシーは、新義豊株式会社（以下「当社」）が提供する「SGH Voice」アプリケーション（以下「本アプリ」）における個人情報の取扱いについて説明します。

最終更新日：2026年2月28日

2. 収集する情報

2.1 音声データ

本アプリは、マイクを使用して音声を録音します。この録音データは、音声認識（Speech-to-Text）処理のためにのみ使用されます。

  • 音声データは端末上で一時的に保持され、処理完了後に自動的に削除されます。
  • 音声データは外部のAI API（OpenAI Whisper、Groq等）に送信されて文字起こしされます。
  • 当社のサーバーに音声データを保存することはありません。

2.2 API キー

ユーザーが入力したAPI キー（OpenAI、Anthropic、Groq、ElevenLabs）は、端末内の暗号化ストレージ（EncryptedSharedPreferences）に安全に保存されます。これらのキーが当社のサーバーに送信されることはありません。

2.3 辞書データ

ユーザーが登録したカスタム辞書および修正ルールは、端末のローカルストレージにのみ保存されます。

3. 情報の利用目的

  • 音声認識および文字起こし処理
  • AI による文章の後処理・校正
  • 繁体字中国語への変換
  • カスタム辞書による認識精度の向上

4. 第三者への提供

音声データは、ユーザーが設定したAPI プロバイダー（OpenAI、Anthropic、Groq）に音声認識・テキスト処理の目的で送信されます。各プロバイダーのプライバシーポリシーについては、各社のウェブサイトをご確認ください。

それ以外の第三者への情報提供は一切行いません。

5. マイクの使用について

本アプリは android.permission.RECORD_AUDIO 権限を使用します。この権限は、ユーザーが音声入力ボタンを押した際の音声録音にのみ使用されます。バックグラウンドでの録音は一切行いません。

6. データの保護

  • API キーは AES-256 で暗号化して保存
  • すべてのAPI 通信は HTTPS で暗号化
  • 音声データは処理後に即座に削除

2. 蒐集的資訊

2.1 語音資料

本應用程式使用麥克風錄製語音。該錄音資料僅用於語音辨識（Speech-to-Text）處理。

  • 語音資料在裝置上暫時保存，處理完成後自動刪除。
  • 語音資料會傳送至外部 AI API（OpenAI Whisper、Groq 等）進行文字轉譯。
  • 本公司不會在伺服器上儲存任何語音資料。

2.2 API 金鑰

使用者輸入的 API 金鑰（OpenAI、Anthropic、Groq、ElevenLabs）以加密方式安全儲存於裝置內（EncryptedSharedPreferences）。這些金鑰不會被傳送至本公司的伺服器。

2.3 詞庫資料

使用者註冊的自訂詞庫及修正規則僅儲存在裝置的本機儲存空間中。

3. 資訊的使用目的

  • 語音辨識及逐字稿處理
  • AI 文字後處理與潤稿
  • 繁體中文轉換
  • 透過自訂詞庫提升辨識準確度

4. 第三方資訊提供

語音資料會傳送至使用者設定的 API 供應商（OpenAI、Anthropic、Groq）以進行語音辨識與文字處理。各供應商的隱私權政策請參閱其官方網站。

除此之外，本公司不會向任何第三方提供資訊。

5. 關於麥克風使用

本應用程式使用 android.permission.RECORD_AUDIO 權限。此權限僅在使用者按下語音輸入按鈕時使用。不會進行任何背景錄音。

6. 資料保護

  • API 金鑰使用 AES-256 加密儲存
  • 所有 API 通訊皆透過 HTTPS 加密
  • 語音資料處理後立即刪除

1. Introduction

This Privacy Policy describes how Shingihou Co., Ltd. ("we", "us") handles personal information in the "SGH Voice" application ("the App").

Last Updated: February 28, 2026

2. Information We Collect

2.1 Audio Data

The App uses the microphone to record audio. This recording data is used solely for speech-to-text processing.

  • Audio data is temporarily stored on the device and automatically deleted after processing.
  • Audio data is sent to external AI APIs (OpenAI Whisper, Groq, etc.) for transcription.
  • We do not store any audio data on our servers.

2.2 API Keys

API keys entered by the user (OpenAI, Anthropic, Groq, ElevenLabs) are securely stored in the device's encrypted storage (EncryptedSharedPreferences). These keys are never transmitted to our servers.

2.3 Dictionary Data

Custom dictionaries and correction rules registered by the user are stored only in the device's local storage.

3. How We Use Information

  • Speech recognition and transcription
  • AI-powered text post-processing and proofreading
  • Traditional Chinese character conversion
  • Improving recognition accuracy through custom dictionaries

4. Third-Party Sharing

Audio data is sent to API providers (OpenAI, Anthropic, Groq) configured by the user for speech recognition and text processing purposes. Please refer to each provider's website for their privacy policies.

We do not share any information with any other third parties.

5. Microphone Usage

The App uses the android.permission.RECORD_AUDIO permission. This permission is used solely for recording audio when the user presses the voice input button. No background recording is performed.

6. Data Security

  • API keys are encrypted with AES-256
  • All API communications are encrypted via HTTPS
  • Audio data is deleted immediately after processing
