# 🎙 Voice Input — 快速安裝指南 (給同事)

這是一個由林紀全 (CEO) 開發/改進的 AI 語音輸入工具，可以完全替代 Typeless。它支援「按住快捷鍵說話 ➔ 自動修正錯字與去除嗯啊 ➔ 自動貼上」，並且支援 **免費的本地 AI 處理 (Hybrid Mode)** 以節省 API 費用。

## 📥 第一步：下載與準備

1. 將收到的 `VoiceInput.zip` 解壓縮。
2. 將解壓縮出來的 `voice-input` 資料夾，移動到你喜歡的地方（例如你的「文件」或「桌面」資料夾）。

## ⚙️ 第二步：前置作業 (安裝 Ollama - 選用但強烈建議)

如果你使用的是 **M系列晶片的 Mac (M1/M2/M3/M4)**，強烈建議安裝本地 AI 模型，這樣日常短句語音輸入將 **完全免費且極快**。

1. 到 [Ollama 官網 (ollama.com)](https://ollama.com/) 下載並安裝 Ollama。
2. 打開 Mac 的「終端機 (Terminal)」。
3. 貼上以下指令並按 Enter，這會下載我們專用的輕量級中文 AI 模型 (Qwen 2.5 3B)：
   ```bash
   ollama run qwen2.5:3b
   ```
   *(下載完成後會出現對話提示字元，輸入 `/bye` 離開即可。)*

## 🚀 第三步：首次安裝與啟動

1. 打開 Mac 的「終端機 (Terminal)」。
2. 輸入 `cd ` (注意後面有一個空格)，然後把剛剛的 `voice-input` 資料夾 **拖曳** 到終端機視窗內，按下 Enter。
   *(這會讓終端機進入該資料夾)*
3. 貼上以下指令並按 Enter 執行安裝：
   ```bash
   ./run.sh
   ```

**第一次執行時會自動：**
- 安裝必要的 Python 環境與錄音套件。
- 跳出 macOS 權限請求。請務必到 **系統設定 ➔ 隱私與安全性** 中，在以下三個地方允許「Terminal (終端機)」：
  1. 麥克風 ✅
  2. 輔助使用 ✅ (這很重要，不然無法自動貼上)
  3. 輸入監控 ✅
- 自動在瀏覽器打開 Dashboard 設定網頁 (http://localhost:7865)。

## 🔑 第四步：設定 API 金鑰

在剛打開的 Dashboard 網頁中，切換到「設定」分頁：
1. 填入 **OpenAI API Key** (必填，用於處理長語音)。
2. 填入 **Anthropic API Key** (必填，用於長句的深度潤稿)。
3. 確認最下方的 **「混合模式 (Hybrid Mode)」** 是否有打開（如果有安裝 Ollama 的話）。
4. 點擊「儲存設定」。

## 🎤 開始使用！

1. 只要右上角選單列有出現 **🎙 圖示**，代表程式正在背景執行。
2. 找個可以打字的地方（LINE、Word、網頁等），**游標點一下**。
3. **按住鍵盤右邊的 `Command (⌘)` 鍵不放**，開始說話。
4. 說完後 **放開按鍵**。
5. 等待 1~3 秒，文字就會自動貼上了！

---

### 💡 日常使用與關閉
- **關閉：** 點擊右上角的 🎙 圖示，選擇「退出」。
- **日常啟動：** 打開終端機，進入 `voice-input` 資料夾，執行 `./run.sh` 即可。
